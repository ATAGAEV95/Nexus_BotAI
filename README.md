## 🚀 Что умеет этот бот

### 1. 🎮 Мониторинг Steam раздач
Основная фича бота. Каждый день в **12:00 по МСК** бот автоматически проверяет RSS-ленту [GamerPower](https://www.gamerpower.com/rss/steam) на наличие новых бесплатных раздач Steam и рассылает их в подписанные Discord-каналы.

*   **Умная дедупликация**: Уже разосланные игры сохраняются в базе данных — повторных уведомлений не будет.
*   **Фильтрация**: DLC и несегодняшние раздачи автоматически отсеиваются.
*   **Ручная проверка**: Администраторы могут вызвать проверку в любой момент командой `?steam`.

### 2. 📢 Подписки на рассылки
Любой Discord-сервер может подписаться на получение ежедневных уведомлений о Steam-раздачах. Подписки хранятся в PostgreSQL и сохраняются между перезапусками бота.

## 🛠 Команды бота

> Команды запускаются с префиксом `?`.

### 👤 Общие команды
| Команда | Описание |
| :--- | :--- |
| `?ping` | Проверить задержку подключения бота |
| `?help` | Показать список всех доступных команд |

### 🛡 Команды Администратора
| Команда | Описание |
| :--- | :--- |
| `?steam` | Немедленно проверить и показать актуальные бесплатные раздачи Steam |
| `?set_steam` | Привязать **текущий канал** к рассылке ежедневных Steam-уведомлений |

> **Примечание**: Команды администратора доступны только участникам с правами администратора сервера или владельцу бота.

## 🏗 Технологический стек

*   **Ядро**: `Python 3.13+` & `discord.py` (Cogs architecture)
*   **База данных**: `PostgreSQL` (SQLAlchemy + asyncpg) — хранение подписок и истории раздач
*   **Планировщик**: `APScheduler` (AsyncIOScheduler) — ежедневные задачи по расписанию (МСК)
*   **HTTP**: `aiohttp` — асинхронные запросы к RSS-лентам
*   **Инфраструктура**: `Docker`, `uv` (пакетный менеджер), `Ruff` (линтинг)

## 📥 Полное руководство по установке

### Шаг 1: Скачивание и подготовка

1.  Убедитесь, что у вас установлены [Docker](https://www.docker.com/) и [Git](https://git-scm.com/).
2.  Склонируйте репозиторий:
    ```bash
    git clone https://github.com/ATAGAEV95/Nexus_BotAI.git
    cd Nexus_BotAI
    ```
3.  Создайте файл конфигурации из шаблона:
    ```bash
    cp .env.example .env
    ```

### Шаг 2: Настройка конфигурации (.env)

Откройте файл `.env` и заполните переменные:

| Переменная | Описание |
| :--- | :--- |
| `DC_TOKEN` | Токен бота из [Discord Developer Portal](https://discord.com/developers/applications) |
| `DATABASE_URL` | URL подключения к PostgreSQL (формат: `postgresql+asyncpg://user:pass@host:port/db`) |

### Шаг 3: Сборка и запуск

#### 🐳 Через Docker (рекомендуется)

1.  **Сборка образа**:
    ```bash
    docker build -t nexus-bot-ai .
    ```
2.  **Запуск контейнера**:
    ```bash
    docker run -d --restart=always --name nexus-bot-ai --env-file .env nexus-bot-ai
    ```
